# Build a Scene with Map, Encounter, and Music

Use this workflow when a scene needs table-ready support.

## 1. Create the Scene

1. Open **D&D Hub: Open Campaign Home**.
2. Select the active campaign.
3. Select **Create Content**.
4. Choose **Scene**.

Keep the required fields minimal first. Use advanced fields only when the scene needs more structure.

![Create Content](assets/usability/create-content.png)

## 2. Add or Link a Map

Open **Map Manager** from Campaign Home, or use map prompts from the scene/session dashboard.

Use maps for tactical spaces, exploration layouts, or visual anchors. Projection remains explicit from the GM map view so a map is not shown to players by accident.

## 3. Build the Encounter

Use **Build Encounter** from Campaign Home. The encounter builder uses the party selected in the active campaign whenever possible.

After saving the encounter, link it to the scene or start it from the Session Run Dashboard.

## 4. Add Music and SFX

Open **Music Settings** from **Settings** -> **D&D Campaign Hub** -> **Audio & Projection** to choose the audio folder, playlists, and soundboard entries.

Use the music player for live playback. Use inline SFX controls when a sound cue belongs directly in the session text.

## 5. Test the Scene

Before play:

1. Open **D&D Hub: Start Session**.
2. Navigate to the scene.
3. Confirm the map, encounter, music, SFX, and handout prompts look correct.
4. Trigger only what you want to test.

The goal is a scene that can be run from one dashboard without hunting through folders.
