# Recover or Continue a Session

Use this workflow when returning to a campaign after a break.

## Continue From Campaign Home

1. Run **D&D Hub: Open Campaign Home**.
2. Select the active campaign.
3. Select **Continue Last Session**.

The plugin opens the latest session note first. It then looks for the most relevant linked adventure scene. In-progress scenes are preferred over completed scenes because they are usually where play stopped.

If no linked scene is available, the plugin opens the linked adventure. If no adventure is linked, it opens only the latest session note.

## What to Check

Before restarting play, review:

| Area | Check |
| --- | --- |
| Session note | Last outcome, unresolved questions, next hook |
| Scene | Current status and links |
| Party | Current roster and resources |
| Encounter | Whether combat is active or already resolved |
| Music/maps | Whether the linked controls still point to the right assets |

## If the Wrong Campaign Opens

Open Campaign Home and select the desired campaign from the dropdown. Creation and continuation actions should follow that selected campaign.

If a modal still shows another campaign, close it and reopen it from Campaign Home so it receives the active campaign context.
