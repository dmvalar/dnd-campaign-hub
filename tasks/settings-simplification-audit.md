# Settings Simplification Audit

## Goal

Make settings easier to scan by grouping controls around GM workflows instead of plugin internals.

## Changes

- Added a top-level Setup section with wizard, checklist reset, Campaign Home, and Create Content shortcuts.
- Added an Audio & Projection section for music settings, live music player, and projection hub access.
- Renamed Battle Maps & Combat to Maps & Encounters and clarified the map/combat descriptions.
- Renamed SRD Data Import to Reference Data so optional imports read as supporting content.
- Clarified migration wording and emphasized automatic backups before file changes.

## Follow-ups

- Consider moving deeper music-player configuration into dedicated subtabs inside the music modal rather than exposing more settings here.
- Consider adding a compact "current active campaign" display once settings has a shared campaign selector component.
